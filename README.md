# Large Language Model Project

This project explores language modeling in Jupyter using PyTorch.

## Project Overview
I built and analyzed a transformer-based text generation model and compared its output to a pre-trained GPT-2 model.

## Methods
- Implemented a decoder based transformer architecture
- Generated text from a custom prompt
- Compared custom model output to GPT-2
- Calculated model parameters and examined architecture behavior

## Tools Used
- Python
- Jupyter Notebook
- PyTorch
- Hugging Face Transformers
